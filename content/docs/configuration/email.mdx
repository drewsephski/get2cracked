## Email Configuration

This guide provides simplified instructions for setting up email services using Resend for your Get Cracked Starter SaaS boilerplate.

<Steps>

### 1. Create a Resend Account

If you don't already have one, create an account on Resend [here](https://resend.com/signup).

### 2. Obtain Your Resend API Key

After signing in to your Resend account, you will be prompted to create your first API key. Copy this key.

### 3. Configure Your `.env` File

Paste your Resend API key into your project's `.env` file:

```dotenv
RESEND_API_KEY=re_your_resend_api_key
```

</Steps>
