## ðŸ†˜ Migration Guide

For existing users upgrading to `v0.4.0`, please follow these steps:

1. **Run Database Migrations**: Execute `pnpm run db:push` to apply the latest schema changes.
2. **Update Environment Variables**: Review and update your environment variables, especially if utilizing new features.
3. **Regenerate Prisma Client**: Run `pnpm run db:generate` to ensure your Prisma client is up-to-date with the new schema.
4. **Clear Cache and Restart**: Clear your application cache and restart your development server to apply all changes.

All existing data and configurations will be preserved during this upgrade process.
